## マークダウンとは

みなさんはマークダウンというものをご存知でしょうか。私はこの業界に入ってつい最近その存在を知ったのですが、ブログ作成やＩＴ系の業種の人たちの中ではわりと一般的なものなようです

簡単に説明するとHTMLなどに代表されるマークアップ言語のお友達です。
では具体的にマークダウンを使用する場合、何が利点になるのでしょうか。

- 素早く書ける。
- HTMLに変換できる。
- エディタが豊富である。

素早く書け、HTMLに変換できるというのは大きな利点でしょう。
また、もともと対応しているわけでなくても、プラグインで対応させることができるなど、いろいろなツールからリアルタイムで表示して編集できるというのも利点の一つなのではないでしょうか。
